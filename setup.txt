FAIR-Agent Setup Instructions
=============================

Follow these steps to set up and run the FAIR-Agent project.

Prerequisites
-------------
1. Python 3.10 or higher
2. Ollama installed (https://ollama.ai/)

Installation Steps
------------------

1. Clone the repository (if you haven't already):
   git clone https://github.com/somesh-ghaturle/Fair-Agent.git
   cd Fair-Agent

2. Install Python dependencies:
   pip install -r requirements.txt

3. Set up Ollama:
   - Start the Ollama server:
     ollama serve
   
   - In a separate terminal, pull the required model:
     ollama pull llama3.2

Running the Application
-----------------------

1. Start the FAIR-Agent server:
   python start_server.py

2. Access the web interface:
   Open your browser and navigate to: http://127.0.0.1:8000

Additional Scripts
------------------
- Run Baseline Evaluation:
  python scripts/run_baseline_evaluation.py

- Compare Baselines:
  python scripts/baseline_comparison_demo.py

- Test Standardization:
  python src/agents/test_standardization.py
